Video Link Downloader

Mac setup in one clear path.

  1. Double-click Install Video Link Downloader.command.
  2. When Chrome opens, turn on Developer mode.
  3. Click Load unpacked.
  4. Select the extension folder from this same package.
  5. Pin the extension and paste a YouTube or Instagram link.
If Mac says Apple could not verify the installer, use this once:
xattr -dr com.apple.quarantine "/path/to/video-link-downloader-mac" Open Terminal, paste the copied command, press Enter, then open the installer again.
